📊 Portfolio de Data Science
Proyectos de análisis de datos y Machine Learning desarrollados durante mi formación como Analista de Datos y estudiante de Inteligencia Artificial.

🚀 Proyectos
🍷 Clasificación de calidad de vinos
Modelos de clasificación para predecir la calidad de vinos a partir de sus propiedades químicas. Se compararon múltiples algoritmos, siendo KNN el de mejor rendimiento con un accuracy de 0.83.
Técnicas: Análisis exploratorio, preprocesamiento, comparación de modelos
Herramientas: Python, Pandas, NumPy, Scikit-learn, Matplotlib

🚢 Predicción de supervivencia — Titanic
Predicción de supervivencia de pasajeros del Titanic comparando tres modelos de clasificación. El Árbol de Decisión fue el de mejor rendimiento con un accuracy de 0.81.
Modelos comparados: KNN, Regresión Logística, Árbol de Decisión
Herramientas: Python, Pandas, Scikit-learn

🔬 Clasificación de tumores — Breast Cancer Dataset
Proyecto integral de clasificación médica que incluye múltiples enfoques y técnicas avanzadas.
Técnicas: Regresión Logística con regularización (L1/L2), SVM, Random Forest, VotingClassifier, K-Means, PCA
Herramientas: Python, Scikit-learn, Seaborn, Matplotlib

🌸 Clasificación y Clustering — Iris Dataset
Clasificación de especies de flores con VotingClassifier combinando tres modelos base, logrando un accuracy de 96.67%. Complementado con clustering K-Means y reducción de dimensionalidad con PCA.
Técnicas: Ensemble (VotingClassifier), K-Means, PCA
Herramientas: Python, Scikit-learn, Matplotlib, Seaborn

📈 Dashboard de gastos corporativos
Análisis de dataset CSV de gastos de viajes corporativos y desarrollo de dashboards interactivos para visualización y toma de decisiones.
Herramientas: Looker Studio, Power BI

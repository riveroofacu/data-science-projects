📊 Portfolio de Data Science
Proyectos de análisis de datos y Machine Learning desarrollados durante mi formación como Analista de Datos y estudiante de Inteligencia Artificial.

🚀 Proyectos
🍷 Clasificación de calidad de vinos blancos
Clasificación binaria para predecir si un vino es de alta calidad a partir de sus propiedades químicas. Se compararon tres modelos con distintos hiperparámetros usando conjunto de validación, siendo KNN (k=15, pesos por distancia) el de mejor rendimiento con un accuracy de 0.83.
Técnicas: Clasificación binaria, tuning de hiperparámetros, validación de modelos
Modelos comparados: Regresión Logística, Árbol de Decisión, KNN
Herramientas: Python, Pandas, NumPy, Scikit-learn

🚢 Predicción de supervivencia — Titanic
Pipeline completo que incluye imputación de valores faltantes con KNN, ingeniería de variables (tamaño de familia, títulos sociales, viaje solo), encoding y normalización. El Árbol de Decisión fue el modelo de mejor rendimiento con un accuracy de 0.81.
Técnicas: EDA, imputación KNN, ingeniería de variables, encoding, normalización
Modelos comparados: Regresión Logística, Árbol de Decisión, KNN
Herramientas: Python, Pandas, Scikit-learn, Seaborn, Matplotlib

🔬 Clasificación de tumores — Breast Cancer Dataset
Proyecto integral de clasificación médica que incluye múltiples enfoques y técnicas avanzadas.
Técnicas: Regresión Logística con regularización (L1/L2), SVM, Random Forest, VotingClassifier, K-Means, PCA
Herramientas: Python, Scikit-learn, Seaborn, Matplotlib

🌸 Clasificación y Clustering — Iris Dataset
Clasificación de especies de flores con VotingClassifier combinando tres modelos base, logrando un accuracy de 96.67%. Complementado con clustering K-Means y reducción de dimensionalidad con PCA.
Técnicas: Ensemble (VotingClassifier), K-Means, PCA
Herramientas: Python, Scikit-learn, Matplotlib, Seaborn

📈 Dashboard de ventas corporativas — Power BI
Reporte interactivo de 3 páginas sobre ventas 2015-2017 (1,85M en ventas, 250K de ganancia). Incluye modelado de datos con 5 tablas relacionadas, KPIs financieros, evolución mensual de ventas, análisis geográfico por región y estado, top clientes y balance de rendimiento por equipo de vendedores.
Técnicas: Modelado de datos relacional, DAX, visualización interactiva, análisis geográfico
Herramientas: Power BI

🌍 Dashboard de ventas globales — Looker Studio
Reporte interactivo de 3 páginas sobre ventas internacionales (1.289M en ventas, 5M unidades, margen promedio 29,64%). Incluye análisis geográfico mundial, rentabilidad por producto, distribución de clientes por edad, top proveedores con saldo pendiente y evolución temporal de ventas 2020-2022.
Técnicas: KPIs financieros, análisis geográfico global, segmentación de clientes, rentabilidad por categoría
Herramientas: Looker Studio
